## BOLUM D: API HATA YONETIMI

### D1. Bos Catch Bloklarini Duzelt (S27)
Yaklasik 18-20 bos catch blogu var. Tum frontend dosyalarinda .catch(() => {}) pattern'ini bul ve duzelt:

Dosyalar: Layout.tsx, Stable.tsx, Shop.tsx, RaceLobby.tsx, RaceBroadcast.tsx, RaceHistory.tsx, Profile.tsx, QuestPanel.tsx, EvolutionModal.tsx

Veri yukleme hatalari icin (useEffect icindeki GET cagrilari):
.catch((err) => { console.error('Failed to load data:', err); toast.error('Failed to load data. Please refresh.'); })

Aksiyon hatalari icin (POST — buy, equip, train, evolve):
.catch((err) => { console.error('Action failed:', err); toast.error('Something went wrong. Please try again.'); })

Layout.tsx balance fetch'leri: sessiz birakilabilir (toast spam'i onlemek icin sadece console.error)
